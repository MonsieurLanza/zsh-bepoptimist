## Installation

Clone or download in your oh-my-zsh custom plugin directory:

    $ git clone git@github.com:sheoak/zsh-bepoptimist.git ~/.oh-my-zsh/custom/plugins/bepoptimist

Add in your plugins list in ~/.zshrc:

    plugins=(
      bepoptimist
    )
